# 微服务

#### 项目介绍
一个小型微服务框架，作为自己对spring cloud的学习实践使用

#### 软件架构
spring cloud ：eurake,config,gateway等
spring-cloud.version：Finchley.RELEASE


#### 使用说明

1. register是注册中心，其他服务都要注册到这里
2. client是服务端
3. configer是配置中心
4. gateway是网关
5. server是测试服务之间调用的服务
6. common是公共依赖代码包
